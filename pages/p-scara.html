Robot de tamaño mediano para el manejo de materiales, ensamble, maquinaria de precisión y aplicación de adhesivos. Con capacidad de Ethernet para controlarlo mediante lenguaje de programación.

Excelente repetibilidad para un montaje de alta precisión y gran capacidad de carga al usar herramientas para atornillar o aplicar adhesivos.Para manejo de materiales, montaje, mecanizado de precisión y aplicación de adhesivos.

  • Capacidad de Ethernet para controlar el robot a través del lenguaje de programación familiar (IEC 61131-3) de Machine Automation Controller NJ / NX / NY Series.
  • Buena repetibilidad para un montaje preciso.
  • Huella mínima con controlador separado.
  • Con alimentación integral y cables de señal.
  • Alcance de 450 mm.
  • Carga útil máxima de 5 kg
  • Peso de 29 a 31 kg.
  • Gran capacidad de carga para usar herramientas y aplicar adhesivos.
